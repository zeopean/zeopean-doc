## nginx 相关

- 查看nginx 进程

```

ps -ef | grep nginx 


```


- 启动 nginx


```

 nginx -c /usr/local/etc/nginx/nginx.conf 

```

- 查看配置文件位置

```

nginx -t  

```

- 查看php-fpm

```
grep php-fpm 

```
